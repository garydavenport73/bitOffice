# bitOffice64

## A suite of cross-platform office apps weighing in under 64kb.

### It contains:

* A personal database with Contacts and Calendar
* A text editor (Notes)
* A word processor (Write) 
* A CSV spreadsheet editor (Tables)
* A calculator
